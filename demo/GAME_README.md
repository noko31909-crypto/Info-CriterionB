# 🎮 WordQuest - English Dictionary Game

A complete, interactive Roblox-style educational game for learning English vocabulary through category-based gameplay.

## ✨ Features

### 🎯 Game Modes
- **Multiple Choice Mode**: Test knowledge with timed questions and 4 answer options
- **Sorting Mode**: Drag-and-drop words into correct categories
- **Category-Based Learning**: Animals, Food, Objects, and Nature

### 🏆 Progression System
- **XP & Leveling**: Earn experience points and level up
- **Coins System**: Collect coins for correct answers
- **Badges**: Unlock achievements like "Animal Master" and "Speed Demon"
- **Streak System**: Track daily learning streaks with fire emoji 🔥

### 📊 Tracking & Stats
- Words learned counter
- Accuracy percentage
- Time played tracking
- Progress bars for each category

### 🎨 UI Features
- Bright, playful Roblox-inspired design
- Smooth animations and micro-interactions
- Glassmorphism effects
- Floating icon backgrounds
- Responsive design (mobile + desktop)

### 🌟 Screens
1. **Main Menu** - Game title, play button, daily challenge, stats
2. **Category Selection** - Choose from 4 categories with progress indicators
3. **Gameplay Screen** - Multiple choice questions with timer and lives
4. **Sorting Mode** - Drag & drop interface with visual feedback
5. **Progress/Profile** - View level, XP, stats, and badges
6. **Leaderboard** - Top players with podium display
7. **Settings** - Sound toggle, language, and difficulty options
8. **Reward Modal** - Stars, coins, and XP earned after completing levels

## 🎮 How to Play

1. **Start**: Click "Play Now" from the main menu
2. **Choose Category**: Select Animals, Food, Objects, or Nature
3. **Select Mode**: 
   - Click OK for Multiple Choice
   - Click Cancel for Sorting Mode
4. **Play**: Answer questions or sort words correctly
5. **Earn Rewards**: Get coins, XP, and unlock badges
6. **Track Progress**: View your stats and achievements

## 🛠️ Technical Stack

- **React 18** with TypeScript
- **React Router 7** for navigation
- **Motion (Framer Motion)** for animations
- **Tailwind CSS v4** for styling
- **Zustand** for state management
- **React DND** for drag-and-drop
- **Radix UI** for accessible components
- **Canvas Confetti** for celebrations
